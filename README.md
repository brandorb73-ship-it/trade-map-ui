# Trade Map UI

- React app
- Route Map and Cluster Graph
- Google Sheets backend
- Login protected

## Setup

1. Copy folder to your PC
2. Run `npm install`
3. Set environment variables:
   - VITE_SHEET_API_URL=YOUR_APPS_SCRIPT_URL
   - VITE_LOGIN_EMAIL=your_email
   - VITE_LOGIN_PASSWORD=your_password
4. Run `npm run build`
5. Deploy to Vercel
